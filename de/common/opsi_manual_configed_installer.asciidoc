== Opsi Configed Installer

Der Opsi Configed Installer ist ein Programm, mit dem Sie schnell und einfach einen {opsi-configed} auf den folgenden Linux-Distributionen installieren können:

* Debian 9, Debian 10, Debian 11,
* openSUSE Leap 15.2, openSUSE Leap 15.3,
* SLES 15 SP1, SLES 15 SP2,
* Ubuntu 18.04, Ubuntu 20.04


[[opsi-configed-installer-download]]
=== Herunterladen und erster Start

Sie können den Configed Installer als ZIP-Datei unter folgendem Link herunterladen:
https://download.uib.de/opsi4.2/stable/configed-installer/ .

Entpacken Sie die Datei und führen Sie 'opsi_configed_installer_archive.sh' aus (durch Doppelklick oder über die Konsole). Warten Sie ein paar Sekunden, bis sich der Installer öffnet.


[[opsi-configed-installer-gui]]
=== Opsi Configed Installer

Es erscheint ein Fenster, in welchem Sie zunächst die gewünschte Sprache für den Configed Installer auswählen können. Klicken Sie dann auf "weiter>" und beantworten Sie die Fragen.

Geben Sie nach den Fragen Ihr Passwort ein und klicken Sie auf "fertigstellen". Einige Sekunden später beginnt die Installation des {opsi-configed}.

.Installation
image::configed-installer_installation.png["Screenshot: Installation", pdfwidth=90%]

Die Installation kann einige Minuten dauern. Am Ende zeigt Ihnen der Configed Installer an, ob sie erfolgreich war.

.Ergebnis
image::configed-installer_success.png["Screenshot: Ergebnis", pdfwidth=90%]

Ist das Ergebnis "success", dann ist Ihr {opsi-configed} betriebsbereit und Sie finden ihn bei Ihren anderen Programmen. Sie können nun den {opsi-configed} starten (siehe Kapitel 4.2 "Login" im opsi Handbuch: https://download.uib.de/opsi_stable/doc/html/opsi-manual-v4.2/opsi-manual-v4.2.html#opsi-manual-configed). +
Beim Ergebnis "failed" können Sie in den Log-Dateien nach dem Fehler suchen oder sich, wenn Sie einen Support-Vertrag haben, gerne auch direkt an uib wenden.
