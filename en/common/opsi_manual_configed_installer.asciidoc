== Opsi Configed Installer

The Opsi Configed Installer is a program to quickly and easily install an {opsi-configed} on the following Linux distributions:

* Debian 9, Debian 10, Debian 11,
* openSUSE Leap 15.2, openSUSE Leap 15.3,
* SLES 15 SP1, SLES 15 SP2,
* Ubuntu 18.04, Ubuntu 20.04


[[opsi-configed-installer-download]]
=== Download and first Start

You can download the Configed Installer as zip-file under the following link:
https://download.uib.de/opsi4.1/misc/helper/ .

Unzip the file and execute 'opsi_configed_installer_archive.sh' (by double click or via console). Wait a few seconds until the installer opens.


[[opsi-configed-installer-gui]]
=== Opsi Configed Installer

A window appears in which you can first select the language for the Configed Installer. Then click on 'next>' and answer the questions.

After the query, type in your password and click 'finish'. A couple of seconds later the installation of the {opsi-configed} will start.

.Installation
image::configed-installer_installation.png["Screenshot: Installation", pdfwidth=90%]

The installation may take some minutes. In the end, the Configed Installer shows you whether it was successful.

.Result
image::configed-installer_success.png["Screenshot: Result", pdfwidth=90%]

If the result is 'success', your {opsi-configed} is ready for use and you can find it among your other programs. You may now start the {opsi-configed} (see chapter 4.2 "Login" in the opsi manual: https://download.uib.de/opsi_stable/doc/html/en/opsi-manual-v4.2/opsi-manual-v4.2.html#opsi-manual-configed). +
If the result is 'failed', you can search in the log files for the error or, if you have a support contract, you can directly contact uib.
