[[opsi-manual-opsiwebgui]]
= {opsi-webgui}

The {opsi-webgui} is a web-based Graphical User Interface for managing the opsi system. 
It simplifies the deployment and management tasks, without installing an application on your device. 
With {opsi-webgui}, you can configure the opsi-servers, set up new opsi-clients, deploy products, inspect logs from any device with a web browser and more.
