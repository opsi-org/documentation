= Tips and tricks
The following chapter describes useful tips for using the {opsi-webgui}.

[[opsi-manual-opsiwebgui-tips-selections]]
== View Selections

All selections are displayed together in the Quick Panel.
In the Quick Selection section, the first tab image:webgui/opsi-webgui_quickpanel_tab_selection.png[Selection tab in the quick panel] shows an editable list of all selections and the number in another tabs indicates the number of servers, clients or products selected. 
Click on this number to view the selections individually.
