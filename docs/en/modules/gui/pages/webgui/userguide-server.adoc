[[opsi-manual-opsiwebgui-opsiservers]]
= opsi-Server
In opsi it is possible to integrate different servers.
The main server is the opsi-Configserver.
All others are called opsi-Depotserver.
(Learn more about xref:server:overview.adoc#opsi-manual-server-overview[opsi-server]). +
After logging into the opsi-webgui, the server selected by default is opsi-Configserver, which is always highlighted in bold.
You can manage server selection by navigating to *Servers > All Servers*.

image::webgui/opsi-webgui_servers.png[Servers]

[[opsi-manual-opsiwebgui-opsiservers-treeselect]]
== Selection using Quick Panel
Alternatively, the server selection can also be managed via the Quick Panel.
Open the Quick Panel by clicking the button in the right corner of the top menu and select the tab image:webgui/opsi-webgui_quickpanel_tab_depots.png[Servers tab in quickpanel].
The number next to the icon shows the number of currently selected servers.
The first tab image:webgui/opsi-webgui_quickpanel_tab_selection.png[Selection tab in quickpanel] shows a quick view of the editable list of all selections.

image::webgui/opsi-webgui_quickpanel_tab_depots_content.png[Content of the server tab in the quickpanel]


[[opsi-manual-opsiwebgui-opsiservers-attributes-parameter]]
== Server Attributes and Parameters (Defaults)
There are two ways to edit the server attributes and parameters:
Either via the *Server > Configuration* page, or *Server > All servers* and the Configuration button image:webgui/opsi-webgui_buttonconfig.png[Configuration].
The latter has the advantage that you can switch between the servers more quickly, as the configuration is displayed next to the servers.
