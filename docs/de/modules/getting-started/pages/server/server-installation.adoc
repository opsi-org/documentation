[[opsi-getting-started-installation]]
= {opsi-Server} Installation

Dieses Kapitel beschreibt die Installation und Konfiguration eines opsi-Servers. Wenn Sie das Kapitel durchgearbeitet haben, sollten Sie einen funktionsfähigen opsi-Server haben. Das ist die Grundlage für alle weiterführenden Kapitel.

NOTE: Wir gehen in den folgenden Kapiteln davon aus, dass Sie eine funktionierende Netzwerkkonfiguration auf Ihrem Server haben (siehe xref:server/requirements[Voraussetzungen]).
