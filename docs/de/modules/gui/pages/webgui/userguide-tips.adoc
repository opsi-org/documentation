= Tipps und Tricks
Im folgenden werden nützliche Tipps zum Umgang mit der {opsi-webgui} beschrieben.

[[opsi-manual-opsiwebgui-tips-selections]]
== Auswahl anzeigen
Alle Auswahlen können gemeinsam im Quick Panel angezeigt werden.

Im Schnellauswahlbereich zeigt der erste Tab image:webgui/opsi-webgui_quickpanel_tab_selection.png[Selection tab in the quick panel] eine editierbare Liste aller Auswahlen und die Zahl in einem weiteren Tab gibt die Anzahl der ausgewählten Server, Clients oder Produkte an. Klicken Sie auf diese Nummer, um die Auswahl einzeln anzuzeigen.
