= Tipps und Tricks
Im folgenden werden nützliche Tipps zum Umgang mit der {opsi-webgui} beschrieben.

[[opsi-manual-opsiwebgui-tips-selections]]
== Auswahl anzeigen
Die innerhalb der Baumstruktur angezeigte Zahl zeigt die Anzahl der ausgewählten Server, Clients oder Produkte an.
Klicken Sie auf diese Nummer, um die Auswahl anzuzeigen.

Alle Auswahlen können zusammen angezeigt werden.
Klicken Sie im oberen Menü auf image:opsi-webgui_buttoninfo.png[Info].
Es zeigt ein Popup mit der Liste der ausgewählten Server und Clients.
