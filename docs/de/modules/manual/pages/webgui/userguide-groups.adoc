= Gruppen-Management

Die Client-Gruppen können auf der Seite *Gruppen* verwaltet werden.
Die für Gruppen verfügbaren Aktionen sind:

- Untergruppe erstellen
- Eigenschaften bearbeiten
- Clients hinzufügen/entfernen
- Gruppe mit Untergruppe und, mit oder ohne Clientzuweisungen entfernen.

Die verfügbaren Aktionen für Clientzuweisungen sind:

- In mehrere Gruppen kopieren
- Client entfernen
